文件操作

  • 启动emacs
    输入如下内容： 

    $emacs  
$emacs -nw

    另外，启动之后，在键盘上键入 q 跳过系统欢迎的信息.

  • 打开目录
    输入 C-x d
  • 查看当前文件所在目录
  • 新建目录
    输入 M-x make-directory
    然后输入路径，回车。
  • 新建文件
    输入 C-x C-f
    然后输入路径，写文件(这样不创建新文件)。
  • 载入一个文件
    输入 C-x C-f
    输入路径，回车，其实打开的是对应文件的一个buffer，每个文件对应一个buffer，buffer名一般就是文件名去掉后缀。
  • 重新载入当前buffer对应文件
    输入
  • 保存文件
    输入 C-x C-s
    输入之后，再输入路径，回车。
  • 全部保存 输入 C-c s
    输入之后，会依次询问每个文件的是否保存，输入 y 表示保存相应文件； n 表示不保存； ! 保存所有; . 保存当前buffer并退出询问; q 直接退出询问；更多的信息可以用 C-h 来查阅。
  • 关闭文件
    输入 C-x k
    输入之后，会进入一个交互命令，输入想要关闭的buffer名字，即可关闭对应的文件再回车，如果什么都不输入直接回车，则默认退出当前buffer关闭对应的文件。
  • 退出编辑器
